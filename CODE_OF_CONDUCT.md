# Code of Conduct

## Standard

jwst-l2-coupled-dynamics follows professional open-source standards. All
participants are expected to engage constructively, respect others'
contributions, and keep discussions focused on the technical work.

## Scope

This standard applies in all project spaces: issues, pull requests, code
review comments, and any other project communication channel.

## Enforcement

Violations may be reported to the project maintainer at
petr.yamyang@gmail.com. All reports will be handled with discretion.
Maintainers reserve the right to remove, edit, or reject contributions that
do not meet this standard.

## Attribution

This project is committed to the principles of the
[Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
